## プログラミングって難しい？

「プログラミングって難しそう...」と思っている方も多いかもしれません。

でも心配はいりません！プロ研では、初心者の方でも安心して学べる環境を整えています。

## まずは何から始める？

### 1. C言語

高専の授業でも学ぶC言語は、プログラミングの基礎を学ぶのに最適です。プロ研では毎年4月に1年生向けのC言語講習会を開催しています。

```c
#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}
```

### 2. Python

初心者にも書きやすく、人気の高い言語です。データ分析やAI開発にも使われています。

```python
print("こんにちは、プログラミングの世界へ！")
```

### 3. Web開発（HTML/CSS/JavaScript）

ウェブサイトを作りたいなら、まずはHTML/CSSから始めましょう。

## プロ研で学ぼう

プロ研では、先輩部員が優しくサポートしてくれます。わからないことがあれば、いつでも質問できる環境があります。

興味がある方は、ぜひ見学に来てください！
