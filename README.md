# 釧路高専プログラミング研究部 4代目公式サイト
Docusaurus上に構築された多分4代目の公式サイト

## 公式サイトの更新
公式サイトの内容は部員各自で更新することができます．

ただし，実際に更新するにはプルリクエストを通じて執行部による承認が必要です．執行部が確認するのは，基本的に公序良俗に反した内容ではないかどうかのみで，内容の検閲は行いません．ただし，ホームに関しては内容の正確性もチェックされます．

### 講習会資料の更新
https://docusaurus.io/docs/docs-introduction を参考にしてください

講習会資料の更新は，その講習会資料を用いる講習会の主催による承認が追加で必要です．

### ブログの更新
今までこのサイトでブログを書いたことがなければ，まず `blog/authors.yml` にあなたの情報を追記してください．

その後，`blog/` 以下に新しいエントリを作成してください．

既存のエントリを更新する際は，過去に自分が書いたエントリであれば自由に更新してもらって構いません（執行部の承認は必要です）．
